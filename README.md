## 🔑 核心结论

- **2024基线**：MT中国工业称重营收约¥18-20亿（基于10-K公开数据推导）
- **三大新支柱**：电池🥇 → 半导体🥈 → 制药🥉 + 压舱石化工数字化 + 期权生物制造
- **五年投入** ~¥10亿 → **增量营收** ~¥16亿 → **ROI** ~1.5x → **回收期** 5-6年
- **2030目标**：工业称重总营收 ¥28.6亿（基线）至 ¥36亿（乐观）
